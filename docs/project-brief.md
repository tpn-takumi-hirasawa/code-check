# Project Brief

## プロジェクト名

code-check

## プロジェクトの目的

Claude Code を活用して、ソフトウェア開発における以下の活動を支援するためのリポジトリ：

- 要求やアイデアの整理
- 機能仕様書の作成
- 技術設計書の作成
- 実装計画の作成
- コードの実装支援
- コードレビュー
- コード改善
- テストの作成と評価
- ドキュメントとコードの整合性確認
- Pull Request の説明文作成

## 利用者（アクター）

- **ソフトウェア開発者**: 要求、仕様、設計、実装、テストにおいて Claude Code の支援を受ける
- **プロジェクト管理者**: Claude Code による開発フローの管理と品質確認

## 対象範囲

### 対象に含む

- Claude Code を活用した開発フローの構築
- カスタムエージェントの定義と実装
- 仕様、設計、実装、レビュー、テストのための標準化されたドキュメント構成
- 各開発工程における標準的なチェックリストと検証方法
